Кантри-стиль

Все мило, душевно, по-простому.

ОБЩЕЕ ОПИСАНИЕ СТИЛЯ.
Кантри стиль – это сельская простота для любителей неприкрытой романтической естественности.
ОПИСАНИЕ ЁЛКИ
Дом, уют, семейный очаг, простота, природа естественность.
ЭЛЕМЕНТЫ
Ёлка украшена с элементами простой деревенской жизни: пряники, дольки апельсинов, корица, мешочки из простой набивной ткани. Простая мешковина. Тона приглушенные, некричащие. Доп. Элементы: мелкий горошек, клетка, цветочный орнамент. Деревянные игрушки.
