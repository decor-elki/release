ЛОЗУНГ
Все мило, душевно, по-простому.
ОБЩЕЕ ОПИСАНИЕ СТИЛЯ.
Кантри стиль – это сельская простота для любителей неприкрытой романтической естественности.
ОПИСАНИЕ ЕЛКИ
Дом, уют, семейный очаг, простота, природа естественность.

Елка украшена с элементами простой деревенской жизни: пряники, дольки апельсинов, корица, мешочки из простой набивной ткани. Простая мешковина. Тона приглушенные, некричащие. Доп. Элементы: мелкий горошек, клетка, цветочный орнамент. Деревянные игрушки.
